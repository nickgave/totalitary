# Totalitary Bot

이 프로젝트는 **디스코드 Anarchy 서버** 전용 봇으로 만들어짐.

## 주요 프로젝트 구조

```
workspace/
├── src/
│   ├── commands/            # 슬래시 커맨드 모듈
│   ├── types/               # 타입 관련 모듈
│   ├── main.ts              # 봇 진입점
│   └── deploy.ts            # 슬래시 커맨드 등록 스크립트
├── package.json
├── tsconfig.json
└── README.md
```
